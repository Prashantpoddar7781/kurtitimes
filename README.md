## Run Locally

**Prerequisites:** Node.js 18.x or higher

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:3000`

## Deploy to Vercel

The app is configured for easy deployment on Vercel:

1. **Push your code to GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/Prashantpoddar7781/kurtitimes.git
   git push -u origin main
   ```

2. **Deploy on Vercel:**
   - Go to [vercel.com](https://vercel.com)
   - Sign in with your GitHub account
   - Click "New Project"
   - Import your repository: `Prashantpoddar7781/kurtitimes`
   - Vercel will automatically detect the Vite configuration
   - Click "Deploy"

3. **Your app will be live!** Vercel will provide you with a deployment URL.

The `vercel.json` file is already configured for optimal Vite deployment. No additional configuration needed!
